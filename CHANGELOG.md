## [0.1.0] - 2024-01-15

### Adicionado
- Estrutura inicial do projeto com Vite e React
- Página Home com apresentação profissional
- Página Projetos com galeria de trabalhos
- Página Contato com formulário
- Componente Header com navegação
- Componente Footer com links sociais
- Componente ProjectCard reutilizável
- Suporte a acessibilidade (WCAG 2.1)
- Metatags e favicon para SEO
- Documentação de contribuição
- Templates para Issues e Pull Requests
- Configuração para deploy em GitHub Pages e Vercel

### Melhorado
- Responsividade em todos os dispositivos
- Performance de carregamento com Vite
- Organização de código com componentes modulares

### Segurança
- Dependências auditadas e atualizadas
